- 我们经常用电脑记笔记，而iPhone壁纸在手机上设置。因此，公式壁纸应用操作较繁琐，且分以下2部分：
	- 电脑上负责生成壁纸的电脑端应用
	- iPhone上负责设置壁纸的快捷指令

# 安装步骤
## 1. 设置云盘
- 在iPhone和电脑上**同时安装并启用**下列之一：
	- iCloud
	- Onedrive
	- Dropbox
	- 其他任何**方便访问**的云盘服务
		- **方便访问的云盘**指的是能够从iPhone文件应用中直接打开其中文件的云盘，而不是像百度网盘那样需要登录（当然如果你折腾好了相应需要鉴权的流程也OK））
- **在云盘中**创建一个空文件夹作为照片存放处

## 2. 安装软件本体
- 用电脑在页面[GitHub - PalaiologosLei/DailyTipsApp · GitHub](https://github.com/PalaiologosLei/DailyTipsApp)的releases一栏下载

## 3. 获取快捷指令
- 用iPhone打开此链接：[快捷指令](https://www.icloud.com/shortcuts/c999d4fac0b140d3a7f0d34d66fe8901)下载
- 下载后，需要手动将快捷指令中的这两处路径（指`DailyTips`）修改为你在步骤1中的创建好的照片存放处
	- （第一项）从路径 `images_index.json` 获取 `DailyTips` 中的文件
	- （倒数第三项）从路径 `来自列表的项目` 获取 `DailyTips` 中的文件
- 修改后，上述快捷指令中，所有原来是`DailyTips`的路径应该指向你的照片存放处。此后无需其他修改（除非你想整点花活（bushi））
- （可选）将快捷指令添加到桌面上
- （可选）设置自动化，并在自动化中调用此快捷指令
# 使用步骤
## 1. 维护笔记库
- 目前，公式壁纸应用将从markdown格式笔记中提取具有如下格式的公式：
``` markdown
- 【事件的独立性】
	- $E(XY)=E(X)E(Y) \Leftrightarrow X, Y$相互独立
	- 独立性和相关性不等价
	- 其他说明
	- ...... 
```
- 即，第1行为公式名称（必须具有中文中括号），第2行为公式本体，第3行至第n行为公式说明。第2行至第n行必须比第1行多一层缩进。第1行可有任意层数的缩进。
- 你可以使用如下提示词，让AI根据现有笔记生成符合上述格式约定的markdown笔记：
``` txt
请你在我上传的文件中提取出所有数学公式和具有数学形式的重要知识点，按照如下格式写成markdown笔记，并输出到单独的代码框中：
- 【公式名称1】
	- 公式本体
	- 说明1
	- 说明2
	- ......
- 【公式名称2】
	- 公式本体
	- 说明1
	- 说明2
	- ......
注意，第1行为公式名称（必须具有中文中括号），第2行为公式本体（必须使用$...$来引用公式，不能为单行公式，只能为行内公式），第3行至第n行为公式说明。第2行至第n行必须比第1行多一层缩进。第1行可有任意层数的缩进，也就是说你可以添加其他的层次结构和文本。

如果文件中的公式数量过大，请你输出一部分，并且告诉我你处理了哪部分、还剩哪部分。
```

## 2. 使用应用进行壁纸生成
- 第一次打开应用以及第一次进行壁纸生成时，应用可能会卡死，无需担心，一会就好了
- 应用中可以调整很多生成设置，相应设置可能带来的效果都比较易懂，这里就不再赘述
- 生成好图片后，打开云盘检查同步情况，在手机上进行测试
